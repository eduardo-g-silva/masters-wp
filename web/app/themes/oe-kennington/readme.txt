Maat or the Improved Bootstrap: the purpose of the theme is to teach how to make a theme.

== Installation ==
Manual installation:

1. Upload the `pinbin` folder to the `/wp-content/themes/` directory

Activiation and Use

1. Activate the Theme through the 'Themes' menu in WordPress
2. See Appearance -> Pinbin Options to change theme specific options
